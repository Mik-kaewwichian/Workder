# WORKDER Monorepo (Web + Mobile + API)

This repository is a production-ready **starter structure** for the WORKDER platform:
- **Frontend Web**: Next.js (separate UX for mobile vs desktop)
- **Frontend Mobile**: Expo React Native (native UX)
- **Backend API**: NestJS (modular, scalable)
- **Worker**: background jobs (queues/cron/webhooks)
- **Shared**: core types/schemas/api-client + UI tokens/components
- **Data**: pricing/seed/legal/fixtures
- **Infra**: docker/nginx/k8s/terraform
- **Docs**: architecture/security/runbook/testing

> This template contains runnable skeleton files and placeholders.
